Configurando seu Firebase

Para sincronizar seus dados na nuvem através do Firebase pelo GitHub Pages, siga estes passos:

  1. Acesse o Firebase Console e crie um projeto gratuito.
  2. No painel do projeto, adicione um Aplicativo Web e copie as credenciais de configuração (o objeto firebaseConfig).
  3. Substitua os dados no código deste arquivo na linha 390-405 pelas suas chaves copiadas.
  4. No menu lateral esquerdo do Firebase, vá em Build > Authentication, selecione Sign-in method e ative a opção Anônimo.
  5. Vá em Build > Firestore Database, clique em criar banco de dados. Nas Regras (Rules), permita leitura/gravação temporariamente para que seu app funcione sem restrições complexas: 
    rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

Dica: Sempre que alterar o código, lembre-se de salvar, commitar e enviar o push (git push origin main) para atualizar sua página no GitHub Pages!
